# Reai App - 项目结构调整说明

## 📋 概述

根据UI设计规范，已成功将Flutter项目从默认模板重构为专业的硬件AI助手应用。采用了白绿黑主色调，建立了完整的主题系统和组件库。

## 🗂️ 新的项目结构

```
lib/
├── main.dart                    # 主入口文件
├── theme/                       # 主题系统
│   ├── app_theme.dart          # 主主题配置
│   ├── colors.dart             # 颜色定义
│   └── text_styles.dart        # 文字样式
├── components/                  # 组件库
│   ├── buttons/
│   │   └── app_buttons.dart    # 按钮组件集合
│   ├── cards/
│   │   └── standard_card.dart  # 卡片组件
│   ├── forms/                  # 表单组件(待实现)
│   └── navigation/             # 导航组件(待实现)
├── constants/                   # 常量定义
│   ├── dimensions.dart         # 尺寸和间距常量
│   └── durations.dart          # 动画时长常量
└── pages/                       # 页面文件(待实现)
```

## 🎨 主题系统

### 颜色配置
- **主绿色**: `#00D474` - AI智能、成功、活跃
- **纯白色**: `#FFFFFF` - 背景主色，简洁纯净
- **深黑色**: `#1A1A1A` - 文字主色，专业稳重

### 字体系统
- **中文字体**: PingFang SC
- **英文字体**: SF Pro Display / Roboto
- **等宽字体**: SF Mono / Roboto Mono

### 间距系统
- 基于8dp网格系统
- 标准间距: 4dp, 8dp, 16dp, 24dp, 32dp, 48dp
