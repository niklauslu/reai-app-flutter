# Reai App - BLE功能使用说明

## 🎯 功能概述

Reai App集成了蓝牙低功耗（BLE）中心设备功能，支持与你的硬件产品进行无线通信。目前支持以下设备：

- **点一机 DYJ (v1)** - 多功能智能硬件开发平台
- **点一机卡片版 DYJ Card** - 紧凑型卡片式开发板
- **ReAI 眼镜 ReAI Glass** - 智能增强现实眼镜

## 📱 快速开始

### 第一步：打开应用
1. 在手机上找到并启动"Reai App"
2. 进入主界面，你会看到4个底部导航标签

### 第二步：进入BLE功能
1. 点击底部导航栏的"硬件"标签
2. 向下滚动找到"BLE设备搜索"卡片
3. 点击该卡片进入BLE设备搜索页面

### 第三步：开始搜索设备
1. 应用会自动请求蓝牙和位置权限
2. 点击"允许"授权权限
3. 确保手机蓝牙已开启
4. 点击"开始扫描"按钮

### 第四步：连接设备
1. 应用会搜索附近的BLE设备
2. 找到你的设备后，点击设备卡片
3. 等待连接建立
4. 连接成功后会显示提示信息

## 🔧 详细操作指南

### 权限设置

#### Android设备
- **蓝牙权限**: 用于扫描和连接BLE设备
- **位置权限**: Android系统要求，用于BLE扫描
- **设置位置**: 设置 → 应用 → Reai App → 权限 → 全部开启

#### iOS设备
- **蓝牙权限**: 用于与硬件设备通信
- **设置位置**: 设置 → 隐私与安全性 → 蓝牙 → 允许Reai App

### 设备准备

#### 点一机 DYJ系列
1. 确保设备已开机
2. 设备处于蓝牙广播模式
3. 设备名称格式为"DYJ_xxx"或"DYJ Card_xxx"

#### ReAI Glass
1. 确保眼镜已开机
2. 处于配对模式
3. 设备名称格式为"ReAI_xxx"或"ReAI Glass_xxx"

### 搜索和连接

#### 搜索设备
1. 在BLE设备列表页面点击"开始扫描"
2. 应用会搜索15秒，显示发现的设备
3. 设备按信号强度排序显示
4. 每个设备显示名称、类型、信号强度

#### 连接设备
1. 点击目标设备卡片
2. 等待连接建立（通常3-5秒）
3. 连接成功显示绿色"已连接"标识
4. 连接失败会显示错误提示

#### 断开连接
1. 点击已连接的设备卡片
2. 应用会断开连接
3. 设备状态变为"未连接"

## 📱 界面说明

### 硬件页面
- **硬件产品卡片**: 展示你的三种硬件产品
- **BLE搜索入口**: 带蓝牙图标的渐变卡片
- **操作提示**: "开始搜索设备"

### BLE设备列表页面

#### 顶部状态栏
- **状态图标**: 蓝牙搜索图标（搜索时旋转）
- **状态信息**: 实时显示当前操作状态
- **扫描指示器**: 扫描时显示进度动画

#### 操作按钮
- **开始扫描/停止扫描**: 控制设备搜索
- **刷新**: 重新开始设备搜索

#### 设备列表
- **设备图标**: 根据设备类型显示不同图标
- **设备名称**: DYJ、DYJ Card、ReAI Glass等
- **版本标签**: v1、DYJ Card、ReAI Glass
- **信号强度**: 强/中/弱，用颜色标识
- **连接状态**: 已连接/未连接指示器

## ⚡ 使用技巧

### 提高设备发现率
1. **距离**: 保持设备在手机5米范围内
2. **环境**: 避免金属物体遮挡信号
3. **干扰**: 关闭其他蓝牙连接
4. **刷新**: 多次扫描以提高发现率

### 连接稳定性
1. **保持距离**: 连接后保持设备在有效范围内
2. **避免干扰**: 远离Wi-Fi路由器等信号源
3. **及时断开**: 不使用时及时断开连接节省电量

### 故障排除
1. **权限问题**: 检查应用权限是否开启
2. **蓝牙问题**: 重启手机蓝牙功能
3. **设备问题**: 重启硬件设备
4. **连接问题**: 重新搜索并连接设备

## 🔍 支持的设备类型

### 点一机 DYJ (v1)
- **识别**: 设备名包含"DYJ"（不含"Card"）
- **颜色**: 绿色主题
- **用途**: 多功能开发平台

### 点一机卡片版 DYJ Card
- **识别**: 设备名包含"DYJ"和"Card"
- **颜色**: 蓝色主题
- **用途**: 紧凑型开发板

### ReAI 眼镜 ReAI Glass
- **识别**: 设备名包含"ReAI"或"Glass"
- **颜色**: 橙色主题
- **用途**: 智能AR眼镜

## ⚠️ 注意事项

### 使用环境
- **仅支持移动端**: Android和iOS设备
- **需要蓝牙**: 设备必须支持BLE 4.0+
- **需要权限**: 确保应用权限已授权

### 设备兼容性
- **点一机系列**: 支持所有版本
- **ReAI Glass**: 支持所有版本
- **其他设备**: 可能无法识别类型

### 隐私安全
- 应用仅搜索和连接你的设备
- 不会收集个人数据
- 连接数据仅用于设备控制

## 🎯 常见场景

### 场景1：首次使用
1. 下载安装Reai App
2. 打开应用，进入硬件页面
3. 点击BLE搜索，授权权限
4. 开始搜索你的设备
5. 找到设备后点击连接

1. 打开应用进入BLE搜索
2. 点击扫描查找设备
3. 连接需要的设备
4. 使用完成后断开连接

1. 扫描会发现多个设备
2. 根据设备类型和名称选择
3. 一次只能连接一个设备
4. 可以随时切换连接不同设备
